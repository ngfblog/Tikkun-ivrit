כלי עזר להעתקה נכונה של טקסט עברי מאתרים ומ-ChatGPT להדבקה ב-Word.

## הבעיה

כשמעתיקים טקסט עברי מהאינטרנט ומדביקים ב-Word, נוצרות בעיות:

- טקסט יוצא LTR במקום **RTL**
- סימני מרכאות `"כך"` הפוכים במקום `״כך״`
- סוגריים `(כך)` בכיוון הלא נכון
- רווחים מוזרים לפני ואחרי מספרים
- תווים בלתי נראים שמבלבלים את Word

## הפתרון

הכלי מעבד את הטקסט לפני ההדבקה ב-Word:

- מוסיף סימן כיוון RTL בתחילת כל שורה עברית
- ממיר מרכאות ישרות למרכאות עבריות ״ ״
- מתקן רווחים בין מספרים לאותיות לרווח קבוע
- מסיר תווים בלתי נראים (BOM, soft-hyphen)
- מסיר תווי bidi שגורמים לסוגריים הפוכים

## בונוס — תיקון מקלדת

הקלדת בטעות באנגלית כשהמקלדת הייתה בעברית?
